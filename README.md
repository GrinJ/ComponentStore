# Component Store
[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/GrinJ/ComponentStore)

This project contain example of usage library @ngrx/component-store

To run the project, type in console:

```shell
npm i
ng serve
```

Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.

# Project structure

This is simple Angular repo which contains an isolated file-uploader component.
You can find it in directory by this path:

```
src/app/file-uploader
```


Example of app usage in ```app.component.html``` and ```app.component.ts```
